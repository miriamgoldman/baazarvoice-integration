== Installation ==

= Traditional Method =
1. Place bazaarvoice-integration.zip to the wp-content/plugins/ directory
2. Extract plugin
3. Navigate to "Installed Plugins" and click "Activate"
4. To view, go to Settings -> Baazarvoice Integration
